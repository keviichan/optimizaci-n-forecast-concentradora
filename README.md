# 🛠️ Optimizador No Lineal de Forecast - Operación Concentradora

Este proyecto presenta un avance funcional para la visualización, simulación y gestión analítica de datos financieros y operacionales en contextos mineros, diseñado según los requerimientos de la asignatura.

## 🚀 Características Clave
* **Proyección No Lineal:** Modelación matemática adaptativa para corregir anomalías en el consumo proyectado de reactivos químicos complejos (Espumante y Colector).
* **Interfaz Interactiva (Streamlit):** Permite a los tomadores de decisiones simular escenarios operacionales en tiempo real mediante un factor de complejidad mineral ($\beta$).
* **Ingeniería de Software:** Código limpio, modular y estructurado bajo control de versiones.

## 📦 Instrucciones de Ejecución Local
1. Clonar el repositorio.
2. Instalar dependencias del sistema: `pip install -r requirements.txt`
3. Desplegar el entorno local: `streamlit run app.py`
